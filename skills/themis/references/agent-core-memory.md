# Agent Core Memory

Each agent must keep a compact memory file.

## Purpose

Core memory preserves:
- role mission
- key arguments
- unresolved questions
- rejected claims
- verified claims
- citation status
- handoff notes
- spawn requests

## Agent memory schema

AGENT:
MISSION:
SCOPE:
KEY_ARGUMENTS:
OPEN_QUESTIONS:
CLAIMS_SUBMITTED:
CLAIMS_REJECTED:
SOURCES_NEEDED:
CONFLICTS:
HANDOFF:
NEXT_ACTION:

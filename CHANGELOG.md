# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-06-19
### Added
- Repositioned Themis as a research-specialist workroom skill.
- Added mandatory Citation Auditor and Peer Reviewer roles.
- Added per-agent core memory templates.
- Added dynamic specialist spawning protocol.
- Added citation hardening and claim ledger workflow.
- Added internal-vs-public output boundary.
- Added validators to prevent internal workroom tags from leaking into clean reports.
- Added research simulation example.

## [0.1.0] - 2026-06-18
### Added
- Initial release of Themis: Structured dynamic workrooms for AI agents.
- Core skill pack under `skills/themis`.
- Reference documentation for protocol, role generation, challenge loops, anti-sycophancy, memory compression, simulation, and installation.
- Asset templates for role cards, role reports, conflict cards, shared state, final reports, and simulation cases.
- Local validation and simulation scripts.
- Example simulation outputs for multiple domains.
- Installation scripts for Codex, Claude Code, Gemini CLI, Antigravity, and OpenCode.
- Unit tests for structure, validation, examples, and simulation.

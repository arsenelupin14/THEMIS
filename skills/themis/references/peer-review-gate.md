# Peer Review Gate

The final report cannot be produced until peer review passes.

## Gate sequence

1. Domain agents submit claims.
2. Citation Auditor verifies claims.
3. Failed claims return to originating agents.
4. Peer Reviewer checks methodology and overclaiming.
5. Master Research approves clean draft.
6. Clean final report is generated without internal labels.

## Peer review verdicts

- PASS
- PASS_WITH_MINOR_REVISIONS
- MAJOR_REVISIONS_REQUIRED
- FAIL_CITATION_INTEGRITY
- FAIL_OVERCLAIMING
- FAIL_METHOD

## Blocking failures

- fabricated citation
- source-title mismatch
- real source used for unsupported claim
- [EVIDENCE] without verified source
- public draft contains internal labels
- claim phrased as fact but only supported as hypothesis
